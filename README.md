This script lets you run terraria 1.4.5.0 server on termux.

For the first time use

``` cd && git clone https://github.com/kane9287/terraria-server-script-termux.git && chmod +x terraria-server-script-termux/tserver && ./terraria-server-script-termux/tserver  ```

To run type 
``` tserver  ```

TESTING ONLY:

``` cd && git clone https://github.com/kane9287/terraria-server-script-termux.git && chmod +x terraria-server-script-termux/tserver-testing-updates && ./terraria-server-script-termux/tserver-testing-updates  ```
